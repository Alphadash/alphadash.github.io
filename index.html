About

Always interested in computers, when I was 14 I had my first experience with programming.
I was playing SA-MP, an online modification for GTA San Andreas which connected players together, but left it up to the players to implement their own gameplay through scripts.
The language chosen was Pawn, a little-known language described by the developer as "C-like".
Writing the code and seeing it come alive within the game sparked a passion in me. I had created something, and players were connecting to my server and interacting with what I had made.
I was hooked.

Another game, this one using Java with the libGDX library.
Made as a school project with a three day deadline, this was the first game I had ever made from scratch. Being unfamiliar with game development and on a tight deadline, the code is very messy and includes a lot of "test" code left as-is when it ended up working.
It is a realtime multiplayer game, with a central server handling all communication between clients. The server is authorative to protect against cheating, but with no delay compensation it works best on local networks.
